# 1. 알고리즘 설명
주어진 문제에 대한 모든 부분 집합을 구하며 백트래킹 한다.

1. `isSelected` 배열에 해당 문제의 선택 여부를 저장하며 모든 부분 집합에 대해 백트래킹 한다.
2. 백트래킹 함수의 인자로 `cnt+1`을 넘겨준다.
3. `cnt`가 문제의 수 `N`과 같아졌을 때 선택된 문제들이 주어진 조건을 모두 만족한다면 `answer`를 1만큼 증가시킨다.
