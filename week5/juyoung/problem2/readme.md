# 1. 알고리즘 설명

두 동전을 동시에 사방탐색하며 범위에서 벗어난 동전이 하나일 때 버튼을 누른 최소 횟수를 갱신해주면 된다.

1. 백트래킹 함수의 인자로 버튼을 누른 횟수, 첫번째 동전의 위치, 두번째 동전의 위치를 넘긴다.
2. 사방탐색을 하기 전에 return 할 상황이라면 return 한다.
	- 버튼을 누른 최소 횟수보다 현재 버튼을 누른 횟수가 많은 경우, 굳이 더 탐색할 필요가 없으다.
	- 버튼이 10번보다 많이 눌린 경우
3. 두 동전의 범위를 체크한다.
	- 두 동전 모두 보드 밖으로 떨어졌다면 그냥 return 한다.
	- 둘 중 하나만 보드 밖으로 떨어졌다면 현재 버튼을 누른 최소 횟수와 현재 누른 횟수 중 더 작은 값으로 최소 횟수를 갱신한다.
4. 두 동전을 상하좌우로 사방탐색한다.
	- 새로 이동하는 위치가 두 동전 모두 벽이라면 continue 한다.
	- 둘 중 하나만 벽이라면 벽으로 가는 동전의 위치는 이전의 위치로 한다.
