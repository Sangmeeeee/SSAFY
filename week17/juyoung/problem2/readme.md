# 1. 알고리즘 설명

1. 같은 모양의 블록이 2x2 형태로 붙은 게 있는지 검사하고, 있다면 지운다.
2. 지워진 부분 위에 있는 블록들을 밑으로 내린다.
3. 지워진 블록이 없을 때까지 1~2 과정을 반복한다.

우선 첫번째 과정에서 같은 모양의 블록을 지울 때는 `board` 배열을 복사한 `copy` 배열을 사용했다.  
검사는 `board`에서, 업데이트는 `copy`에서 해주었다.  
네 칸을 묶어서 한꺼번에 검사하고, 네 칸 모두 같은 블록인 경우만 `copy`의 해당 블록들을 `'0'`으로 업데이트 해주었고,  
`copy` 배열의 해당하는 위치의 블록이 `'0'`이 아닌 경우에 `remove_cnt`를 증가시킨다.  
이 과정에서 만들어진 `copy` 배열을 반환하여 `board`에 덮어쓴다.  

만약 `remove_cnt가 0이라면 더이상 지울 수 있는 블록이 없으므로 종료한다.  
그렇지 않다면 `ansewr` 변수에 `remove_cnt`를 더해주고, 두번째 과정을 계속한다.

두번째 과정에서는 `board` 배열을 이용하여 빈칸이 없도록 블록들을 아래로 옮겨준다.  
컬럼별로 진행하며, 컬럼의 각 행을 돌면서 `'0'`이 아닌 블록만 `temp` 배열에 삽입하고 해당 블록을 `'0'`으로 변경한다.  
모든 행을 돌고 나서, 다시 (m-`temp`의 길이)행부터 `m-1`행까지 `temp`에 든 원소들로 차례로 덮어써준다.
