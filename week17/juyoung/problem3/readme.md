# 1. 알고리즘 설명

1차원 벡터를 이용해서 풀었다.

1. 1차원 벡터 `cache`를 `cacheSize`만큼 빈 문자열로 초기화한다.
2. `cities`에서 도시 이름을 꺼내서 소문자로 변환한다.
3. `cache` 안에 소문자로 변환한 도시 이름이 있는지 확인한다.
	1. 만약 있다면, 
		- 해당 원소를 지운다. 
		- `cache`의 제일 뒤에 push한다.
		- `answer`에 1을 더한다.
	2. 만약 없다면,
		- 첫번째 원소(=가장 오래 참조되지 않은 원소)를 지운다.
		- 제일 뒤에 소문자로 변환한 도시 이름을 push한다.
		- `answer`에 5를 더한다.
