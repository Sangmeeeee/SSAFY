## 문제 이해

### 예외사항1

1 2 3 4 5

와 같이 한쪽의 부호로만 가득 차있을때 문제가 생기기에 예외처리를 해주었다.

### 예외사항2

5
-2 50 -99 -1 -98

이런 경우에 -1, -2 가 되 어야하는데 계속 -1,50 이 뽑히더라. 잘 분석 해보니깐, - + 되는경우에도 시작 하기 전에 가장 작으면서 같은 부호끼리 더했을때 값도 고려를 해주어야했다.
