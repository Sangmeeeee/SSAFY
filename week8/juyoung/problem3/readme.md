# 1. 알고리즘 설명

1. 인접리스트를 만들며 진입 차수를 저장하는 배열 `degree`에 각 정점의 진입 차수를 저장했다.
2. `degree` 배열 값이 0인 정점을 큐에 삽입한다.
3. 큐가 빌 때까지 다음을 반복한다.
	1. 선택된 정점을 큐에서 삭제한다.
		- `q.pop();`
	2. 선택된 정점에 부속된 모든 간선에 대해 진입 차수를 감소시킨다.
		- `degree[next]--;`
	3. 만약 선택된 정점의 진입 차수가 0이 되면
		- 해당 노드를 큐에 삽입한다.
		- `현재 학기+1` 값을 학기 배열의 선택된 정점 인덱스에 저장한다.


# 2. 새로 배운 내용(코드 위주)
## 위상 정렬(Topological Sort)
**어떤 일을 하는 순서**를 찾는 알고리즘  
즉, 방향 그래프에 존재하는 각 정점들의 선행 순서를 위배하지 않으면서 모든 정점을 나열하는 것

관련 예시에는 선수 과목이나 각각의 작업이 완료되어야만 끝나는 프로젝트 등이 있다.

### 위상 정렬 구현하기
1. 진입 차수가 0인 정점(=들어오는 간선의 수가 0개인 것)을 큐에 삽입
2. 큐에서 하나씩 꺼내며 선택된 정점과 이에 부속된 모든 간선 삭제
	1. 선택된 정점을 큐에서 삭제
	2. 선택된 정점에 부속된 모든 간선에 대해 간선의 수를 감소
3. 큐가 빌 때까지 위 과정 반복

따라서 진입 차수를 저장하는 배열, 인접리스트를 저장할 배열이 필요하다.
