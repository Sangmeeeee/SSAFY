# 1. 알고리즘 설명

가장 작은 차이를 구하는 문제이므로 입력 받은 배열을 **오름차순 정렬**한다.
그리고 N-1번째부터 1번째까지 i, i-1번째부터 0번째까지 j를 사용하여 배열 인덱스에 접근한다.

`M` 이상이면서 가장 작은 차이를 구해야하므로 i번째 수와 j번째 수의 차이가
1. `M`보다 작으면 `min_value`를 업데이트하는 것 없이 계속 진행한다.  
2. `M`보다 크거나 같으면 `min_value`를 해당 값으로 업데이트 한다.  

배열이 이미 정렬된 상태이므로 **j가 앞으로 이동할 때마다 i번째 값과 차이가 점점 커진다.**  
따라서 `min_value`를 업데이트했다면, j에 대한 탐색을 끝내고 i를 1만큼 줄인 후 다시 시작해야 한다.  
i를 1만큼 줄이기 이전에, `min_value` 값이 이미 `M`이라면 더이상 진행할 필요가 없으므로 i에 대한 탐색도 종료한다.  
마지막에 `min_value`에 저장된 값이 정답이 된다.
