# 1. 알고리즘 설명

1. queue에 다리의 길이만큼 0을 push한다.
2. queue를 하나씩 pop하고, 그때마다 대기하고 있는 제일 첫번째 트럭이 현재 다리 위에 올라갈 수 있는지*(현재 다리 위 트럭 무게의 총합 + 대기하고 있는 첫번째 트럭의 무게 ≤ 다리가 감당할 수 있는 무게인지)* 검사한다.  
    1. 올라갈 수 있다면 그 트럭의 무게를 queue에 push하고 대기열에서 해당 트럭을 뺀다.
    2. 올라갈 수 없다면 queue에 0을 push한다.
3. 만약 대기하고 있는 트럭이 없다면, 더 이상 push 할 필요 없이 pop만 해준다.
    1. 대기하고 있는 트럭이 있을 때는 push를 할 때마다 초를 세아린다.
    2. 대기하고 있는 트럭이 없을 때는 pop을 할 때마다 초를 세아린다.
