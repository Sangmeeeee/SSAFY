# 알고리즘 설명
플로이드-와샬 알고리즘의 기본 문제이고 밑의 2가지만 유의하면 된다.

2. 마지막에 출력할 때 최단 경로가 없는 경우를 고려해서 0을 출력해줘야 한다.
