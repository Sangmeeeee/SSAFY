# 1. 알고리즘 설명

:rose:
**입력** 시 `location`이라는 벡터에 도시들의 위치 정보를 저장한다.  
동시에 해당 도시가 특별한 도시일 경우 `special` 배열의 해당 인덱스를 `true`로 설정한다.  

:rose:
플로이드 와샬 알고리즘을 적용하기 위해 **인접 행렬을 생성**한다.  
`location` 벡터에서 해당하는 두 정점의 위치를 각각 꺼내어 변수에 저장하고  
이를 이용해 인접 행렬에 도시 간 이동 시간을 저장한다.  
이때, 만약 두 도시가 특별한 도시라면!!  
기존에 저장했던 이동 시간과 텔레포트 시간 중 더 작은 값을 인접 행렬에 저장한다.

:rose:
3중 for문을 통해 **플로이드 와샬 알고리즘을 수행**한다.  
인접 행렬을 반복적으로 갱신하면서 최단 시간을 저장한다.  

:rose:
입력되는 두 도시 A, B에 대해 인접 행렬에서 최소 이동 시간을 **출력**한다.
