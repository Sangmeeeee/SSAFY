# 1. 알고리즘 설명

한 지점에서 출발을 하여서 다시 출발 지점으로 돌아왔을 때  
출발을 했을 때보다 시간이 되돌아가 있는 경우가 있는지 없는지 찾는 문제  
= **음수 간선 사이클이 존재하는지, 아닌지 판별하는 문제**

:rose: **입력**  
도로는 양방향 이동이 가능하고 웜홀은 단방향 이동만 가능하다.  
따라서 간선을 저장하는 배열 `edge`에 도로는 양방향을 전부 삽입하고 웜홀은 단방향만 삽입한다.  
또한 웜홀은 도착 시 시간이 뒤로 가게 된다고 했으므로  
이동 시간 `T`를 `edge`에 삽입할 때 `-T` 값으로 넣어야 한다.

:rose: **벨만포드 알고리즘**
출발지는 1번 지점으로 가정했으며, 출발지에서 출발하여 다시 출발지로 돌아왔을 때  
시간이 되돌아가 있다는 것은 음수 간선 순환이 존재한다는 의미이다.  
따라서 벨만포드 알고리즘을 통해 음수 간선 순환 여부를 파악한다.
