# 알고리즘 설명
트리의 조건인 간선의 갯수 = 정점의 갯수 - 1인 것을 이용했다.  
한 정점을 기준으로 BFS로 탐색할 때의 과정이다.  
맨 처음 정점의 갯수 n = 1이고, 간선의 갯수 e = 0이다.
1. 다음 노드가 방문하지 않은 노드면 큐에 추가하고 n++를 한다.
2. 현재 정점에 연결된 노드를 검사할 때마다 e++를 한다.
3. 탐색이 끝나고 e / 2 == n - 1이면 트리이다.
