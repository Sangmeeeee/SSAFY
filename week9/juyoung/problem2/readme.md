# 1. 알고리즘 설명

첫번째 수열의 길이 * 두번째 수열의 길이만큼의 `dp` 배열을 만든다.  
이때 `dp[i][j]`는 **수열1의 0~i 부분수열과 수열2의 0~j 부분수열의 LCS의 길이**이다.

수열1의 0~i 부분수열과 수열2의 0~j 부분수열을 각각 비교하면서 `dp` 배열을 채운다.  
1. 만약 수열1의 n번째 문자와 수열2의 n번째 문자가 같다면
	- `dp[i][j] = dp[i-1][j-1]+1`
	- 이전 부분 수열까지의 LCS 길이에 +1
2. 다르다면
	- `dp[i][j] = max(dp[i-1][j], dp[i][j-1])`
	- 왼쪽 값과 위쪽 값 중 더 큰 값을 대입

`dp` 배열의 마지막 원소가 답이 된다.
