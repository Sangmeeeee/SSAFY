참조 url :


# 1. 알고리즘 설명

1. 명령에 해당하는 톱니바퀴를 기준으로 왼쪽, 오른쪽 각각 맞물려있는 톱니의 극이 같거나 톱니의 범위를 벗어날 때까지 반복문을 돌린다.
	- 몇 번까지 돌려야 하는지를 왼쪽은 `prev`, 오른쪽은 `next`에 저장한다.
2. 명령에 해당하는 톱니바퀴의 이전 톱니바퀴부터 prev까지, 이후 톱니바퀴부터 next까지 회전시킨다.
3.  명령에 해당하는 톱니바퀴를 회전시킨다.

몇 번까지 돌려야 하는지를 저장한 후 한꺼번에 회전시키는 이유는, 하나씩 회전시키다 보면 다른 톱니바퀴와 맞물리는 톱니가 기존과 달라져서 회전시켜야 할 것을 회전시키지 못 하고, 회전시키지 말아야 할 것을 회전시키게 되기 때문이다.
