# 알고리즘 설명
매 시행마다 크기가 가장 작은 2개를 알아야 하기 때문에 우선순위 큐가 필요하다  
큐의 상위에 있는 2개를 pop해서 계산한 다음 다시 큐에 넣는 과정을 반복하면 된다.  
종료 조건은 2가지가 있다.  
1. 큐의 최상단 숫자가 K보다 크거나 같은 경우
- 이때는 계산 횟수를 출력하면 된다.
2. 큐의 사이즈가 2보다 작은 경우
- 큐에 있는 숫자가 K보다 작으면 -1을 출력해야 한다.
