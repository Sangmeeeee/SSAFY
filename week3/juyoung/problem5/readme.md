# 1. 알고리즘 설명

백준 1715 카드 정렬하기, 13975 파일 합치기 3과 유사한 문제였다.

1. 주어진 스코빌 배열의 원소들을 우선순위 큐에 삽입한다.
2. 우선순위 큐의 front값이 Leo가 원하는 스코빌 지수가 될 때까지
	1. 가장 앞쪽 원소 두 개(`a`, `b`)를 pop하고
	2. `a + b*2`를 다시 우선순위 큐에 push한다.
	3. `answer`를 1만큼 증가시킨다.
