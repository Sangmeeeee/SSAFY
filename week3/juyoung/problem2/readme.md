# 1. 알고리즘 설명

1715 카드 정렬하기와 매우 유사한 문제였다.

**우선순위 큐**를 통해 입력 받고, 큐의 가장 앞 원소 두가지를 pop하여 더한 후 더한 값을 다시 우선순위 큐에 넣어서 큐에 원소가 1개만 남을 때까지 반복한다.

각 원소를 더할 때 그 값을 answer 변수에 더해주고 마지막에 answer 변수의 값을 출력했다.
