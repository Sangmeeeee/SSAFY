# 1. 알고리즘 설명

**우선순위 큐**를 통해 매순간 가장 작은 값 2개가 더해지도록 했다.

1. 오름차순으로 정렬하는 우선순위 큐에 입력 받은 카드의 장수를 저장한다.
2. 큐를 2번 pop하여 나온 두 수를 더한 후, 더한 값을 큐에 다시 push한다. 이때 `answer` 변수에도 두 수를 더한 값을 더해준다.
3. 큐에 원소가 1개만 남을 때까지 위 과정을 반복하다가 원소가 1개만 남으면 `answer` 변수를 출력한다.
