# 1. 알고리즘 설명

다음의 두 단계로 간단히 풀 수 있는 문제였다.  
1. 입력 시 파티에 참여하는 첫번째 사람과 나머지 사람들을 연결한다.
2. 모든 파티에 대해 진실을 아는 사람이 있는지 검사한다.

1번은 `union()`과 `find()` 함수를 통해 수행한다.  
하나의 파티에 참여하는 사람들의 루트를 그 파티에 첫번째로 참여하는 사람의 루트로 변경하다보면  
여러 파티에 걸쳐서 한 번이라도 같은 파티에 참여하는 사람들은 전부 하나의 그래프로 연결된다.

이 상태에서 2번을 수행한다.  
파티 전체를 탐색하며, 진실을 아는 사람의 개수만큼 반복문을 돌린다.  
만약 파티에 참여하는 첫번째 사람과 진실을 아는 사람의 루트가 같다면  
그 사람은 진실을 아는 사람이므로 해당 파티에서는 과장된 내용을 말할 수 없다.  
처음 `M`으로 초기화했던 `answer` 변수를 이 경우에만 1씩 감소시키면  
반복문이 종료되었을 때 `answer` 변수에 정답이 저장된다.
