# 1. 알고리즘 설명

문제에서 각 도시에 대한 연결 정보가 인접 행렬로 주어진다.  
인접 행렬을 입력 받을 때 행 `i`와 열 `j`에 대해 두 도시가 연결되어 있다면  
`Union()`을 통해 두 도시의 루트를 더 작은 값을 기준으로 갱신해준다.

이후 여행 계획 중 첫번째 도시의 루트 노드를 `r`에 저장한다. 
그리고 여행하려는 도시들을 쭉 탐색하며 각 도시의 루트 노드가 `r`과 다르다면  
첫번째 노드와 해당 노드가 연결되어 있지 않다는 의미이므로 계획대로 여행하는 것이 불가능하다.  
이러한 정보를 `isAvailable` 변수에 담고 변수의 값에 따라 `YES`나 `NO`를 출력한다.
