# 1. 알고리즘 설명

다익스트라 알고리즘을 적용하여 간단히 풀 수 있었다.

1. 주어진 간선을 통해 방향 그래프를 만든다.
2. 각 정점까지의 최단 거리를 저장하기 위한 배열 `dist`를 충분히 큰 값으로 초기화한다.
3. 다익스트라 알고리즘을 수행한다.
	1. 우선순위큐가 빌 때까지 각 정점과 인접한 정점들의 최단 거리를 업데이트하고 큐에 push한다.
4. `dist` 배열의 값들을 출력한다.
