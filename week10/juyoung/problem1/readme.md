# 1. 알고리즘 설명

다익스트라 알고리즘을 적용하면 간단하게 풀 수 있는 문제였다.

첫째, `solution()`의 인자로 넘어온 2차원 벡터 `road`에 대해 **인접리스트를 생성**한다.

둘째, 최단 시간 배열 **`dist`을 충분히 큰 수로 초기화**한다.

셋째, 1번 음식점을 시작으로 **다익스트라 알고리즘을 수행**한다.  
시작 음식점에서 시작 음식점까지의 시간이 0, 시작 음식점의 번호가 1이므로  
우선순위큐 `pq`에 `{0, 1}`를 저장하고 `dist[1]`을 0으로 갱신한다. 
그리고 `pq`가 빌 때까지 `pq`에서 하나씩 pop하며 인접한 음식점에 대한 최소 시간을 계산한다.  
이때 인접한 음식점에 대한 최소 시간 값이 갱신되면 `pq`에 `{-갱신된 값, 해당 음식점 번호}`를 push한다.  

넷째, 위 과정이 끝나면 1번 음식점부터 N번 음식점 중  
최소 시간이 K 이하인 음식점의 수를 세아린 것이 정답이 된다.
