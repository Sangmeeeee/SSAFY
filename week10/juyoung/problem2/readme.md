# 1. 알고리즘 설명

problem1인 프로그래머스 배달과 비슷한 문제였다.  
다익스트라 알고리즘을 적용시키면 쉽게 풀 수 있었다.

1. 입력을 통해 **양방향 그래프를 생성**한다.
2. 직원의 수만큼 **다익스트라 알고리즘을 수행**한다.
	1. `dist` 배열을 초기화한다.
	2. 다익스트라 알고리즘을 수행한다.
	3. KIST나 씨알푸드에 도달할 수 없다면 각 `dist` 배열 값을 -1로 갱신한다.
	4. `answer`에 `dist[A]+dist[B]`를 누적해서 더한다.

다익스트라 알고리즘은 problem1과 마찬가지로  
우선순위큐를 사용하여 큐가 빌 때까지 `dist` 배열에 최단 거리를 저장하고  
최단 거리가 갱신될 때마다 해당 정점과 거리 값을 큐에 삽입한다.
